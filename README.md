# 📡 SPU Morse Code Training Station & Dashboard (ระบบฝึกอบรมมอร์ส และ แผงควบคุมระบบ)

ยินดีต้อนรับสู่ **SPU Morse Code Training Station** — แพลตฟอร์มการเรียนรู้และฝึกฝนรหัสมอร์สแบบครบวงจร ที่พัฒนาด้วย **React 19, TypeScript, Tailwind CSS v4, Framer Motion** และเสริมความทนทานของข้อมูลด้วย **Firebase Firestore** เพื่อให้สถิติ ความสำเร็จ และเลเวลของผู้ใช้งานซิงค์แบบเรียลไทม์ข้ามอุปกรณ์ได้อย่างมีประสิทธิภาพ

---

## 🌟 คุณสมบัติเด่น (Core Features)

1. **Campaign Gameplay (โหมดด่านการเรียนรู้)**
   - ด่านการเรียนรู้ออกแบบเป็นภารกิจหลักสถิติมอร์ส (เช่น สัญลักษณ์พื้นฐาน, ตัวเลข, สัญญาณย่อสากลระดับสากล)
   - มีการประเมินคะแนน ความเร็ว และความถูกต้องแบบทันที
2. **Custom Practice (ฝึกฝนตามใจชอบ)**
   - กำหนดข้อความภาษาอังกฤษ ตัวเลข หรืออักษรพิเศษในการเล่นได้อย่างอิสระ
   - ปรับแต่งค่าความเร็วส่ง (WPM) และความถี่เสียง (Frequency) ได้ตามความต้องการของผู้เรียน
3. **Randomized Decoding Drills (โหมดทดสอบจับเวลามอร์ส)**
   - ระบบจำลองการฟังเสียงสุ่มและถอดรหัสเพื่อฝึกฝนประสาทสัมผัสในการรับฟังสัญญาณมอร์ส
4. **Free Tapping Simulator (เครื่องเคาะรหัสอิสระ)**
   - หน้าเคาะมอร์สด้วยแป้นพิมพ์ Spacebar หรือการสัมผัสจำลองคีย์ส่งวิทยุสมัครเล่นจริง
   - ถอดรหัสอักขระที่เคาะออกมาแสดงผลบนหน้าจอสดๆ พร้อมบันทึกข้อความไว้ฝึกซ้ำ
5. **Real-time Firestore Users Sync**
   - ลงทะเบียนสมาชิกและเข้าใช้ได้ทั้ง Google Sign-In ด้วย SPU Google Mail หรือการลงทะเบียนบัญชีปกติทั่วไป
   - บันทึกสถิติจำนวนครั้งในการเคาะรหัส (Total Taps) และความสำเร็จ (Achievements) แบบเรียลไทม์

## 🛡️ แผงควบคุมแอดมินหลังบ้าน (Admin Control Panel)

เมื่อล็อกอินเข้ามาด้วยบัญชีของ **กิตติภัทร ชินวงศ์ (Admin)** จะปรากฏแท็บเมนูพิเศษขึ้นมาด้านบนสุด: **"แผงควบคุมแอดมิน (Admin)"**

แผงควบคุมมีฟีเจอร์การจัดการขั้นสูงดังนี้:
1. **ดูทำเนียบสมาชิกแบบเรียลไทม์ (Live User Directory)**
   - ตรวจสอบรายชื่อสมาชิกทั้งหมดที่ลงทะเบียนในฐานข้อมูล มีแถบค้นหา (Search) ค้นหาจากชื่อ อีเมล หรือ Callsign ได้ทันที
2. **ระบบสลับสถานะต้องการข้อมูลลงทะเบียนใหม่ (Force Re-Registration)**
   - สลับสิทธิ์ผู้ใช้ที่ข้อมูลใบอนุญาตหมดอายุ หรือต้องการคีย์ข้อมูลอัปเดตใหม่
   - สามารถพิมพ์ระบุระบุเหตุผลเพื่อส่งข้อความแจ้งเตือนสีแดง (Alert Banner) ไปยังหน้าหลักของผู้ใช้คนนั้นๆ ให้ลงทะเบียนแก้ไขได้ทันทีเมื่อเข้าสู่ระบบ
3. **ระบบแก้ไขสัญญาณเรียกขานโดยตรง (Modify Principal Callsign)**
   - แอดมินสามารถคลิกปุ่ม **"แก้ Callsign"** เพื่อพิมพ์อัปเดตหรือลบสัญญาณเรียกขานประธาน (Main Primary Callsign) ของสมาชิกได้ทันที โดยจะซิงค์ขึ้นระบบ Firestore แบบเรียลไทม์
4. **ส่งออกรายงาน Google Sheets (Export to CSV Report)**
   - ปุ่มส่งออกข้อมูลสมาชิกในรูปของไฟล์ CSV ที่ผ่านการจัดรูปแบบฟิลด์อย่างสมบูรณ์แบบ ทั้งลำดับที่, ชื่อ-นามสกุล, อีเมล, สัญญาณเรียกขาน, วันที่สมัคร และสถานะของข้อมูล เพื่อนำไปประยุกต์ใช้อัปโหลดเข้า Google Sheets เป็นฐานข้อมูลกลางได้ทันที

## 🛠️ โครงสร้างและการติดตั้ง (Development & Deployment)

แอปพลิเคชันนี้ทำงานบน React 19 และขับเคลื่อนโดย Vite

### คำสั่งใช้งานภายในโปรเจกต์:

* **ติดตั้ง Dependencies ทั้งหมด:**
  ```bash
  npm install
  ```
* **เปิดโหมดพัฒนาเครื่องเซิร์ฟเวอร์หลัก (Local Server - Port 3000):**
  ```bash
  npm run dev
  ```
* **ตรวจสอบแก้ไข Typescript & Linting:**
  ```bash
  npm run lint
  ```
* **การสั่ง Build เพื่อนำไปใช้บนโฮสติ้งจริง:**
  ```bash
  npm run build
  ```
